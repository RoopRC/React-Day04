# 📘 Rules of JSX – React Components Mini Project  

## 📌 Overview  
This project is a simple React application that demonstrates the **Rules of JSX** using a **component-based structure**. It includes Header, Content, and Footer components to build a clean UI.

---

## 🎯 Objective  
- Understand the rules of JSX  
- Learn component-based architecture in React  
- Practice structuring a React project  

---

## 🛠️ Technologies Used  
- React JS (Vite)  
- JavaScript (ES6)  
- HTML5  
- CSS3  

---

## ⚙️ Features  
- Displays JSX rules in list format  
- Uses reusable components  
- Uses semantic HTML (`<nav>`, `<ul>`)  
- Displays React logo using import  
- Styled with CSS

## 💻 Code Explanation  
- `createRoot()` → renders app to DOM  
- `Header.jsx` → logo + title  
- `Content.jsx` → JSX rules list  
- `Footer.jsx` → footer text  
- JSX used to build UI
